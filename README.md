# MILO Files

## 📁 Folder Structure

### ✅ Final Modular Files
| File                   | Description |
|------------------------|-------------|
| `index.html`           | The homepage of the MILO app. Introduces the Death Literacy Questionnaire and features a prominent “Start Quiz” button, visible only to logged-in users. Contains site navigation and links to other sections.|
| `loginSignUp.html`     |Authentication page where users can log in or sign up using Firebase Authentication. |
| `profileSetup.html`    | Profile completion form shown after user sign-up. Asks for demographic and personal context such as location, age group, gender and health status.|
| `quiz.html`            | The main quiz interface containing 29 questions. Responses are submitted to Firestore and used by the backend AI model to generate feedback. Only accessible to logged-in users. |
| `results.html`         | Displays a radar chart and personalized feedback generated by the AI model.  |
| `learn.html`           | Educational hub with trusted external and internal resources. Covers topics such as palliative care, hospice care, advance care planning, grief, communication and Australian support services. |
| `privacy.html`         |Provides tips on protecting personal data and outlines Australian data privacy rights. Features official links to government privacy offices and best practices for data handling.|
| `terms.html`           | Outlines the app’s terms of service including user responsibilities and contact information. Provides transparency about the conditions for using the MILO platform.|
| `app.py`               | The app.py file is the backend server script for the Death Literacy Quiz web application. It is built using the Flask framework and serves as a secure API that generates personalized feedback based on user quiz responses stored in Firebase Firestore. |
| `data.py`              | Contains user responses, question-to-topic mapping, and benchmark thresholds. |
| `feedback.py`          | Stores personalized feedback messages for each topic and score range (low, similar, high). |
| `logic.py`             | Functions to calculate scaled mean scores and generate appropriate feedback. |

## 🔧 Run Locally (Windows, macOS, Linux)

### 1) Prerequisites
- Python 3.9+ installed

### 2) Create and activate a virtual environment
Windows (PowerShell):
```bash
cd M.I.L.O
python -m venv venv
venv\Scripts\Activate
```
macOS/Linux (bash/zsh):
```bash
cd M.I.L.O
python3 -m venv venv
source venv/bin/activate
```

### 3) Install backend dependencies
Install from the requirements file:
```bash
pip install -r Requirement.txt
```

### 4) Run the backend (Flask)
From the `M.I.L.O` directory
```bash
python app.py
```
The API will be available at `http://127.0.0.1:5000`.

### 5) Open the frontend
open `M.I.L.O/index.html` directly in your browser.

### 6) Use the app
1. Sign in on `loginSignUp.html`
2. Take the quiz at `quiz.html`
3. You’ll be redirected to `results.html` where personalized feedback and the radar chart are shown.

### Troubleshooting
- DefaultCredentialsError: ensure you start the backend from `M.I.L.O` so `firebase-app-key.json` is found
